6 Common Mistakes to Avoid

  • FileNotFoundError: The image or CSV file is not in the same folder as the notebook. Move them into the same directory.
  • Forgetting .copy(): Modifying an array or DataFrame without copying it first will change the original. Always use .copy() when the instructions say to.
  • Running cells out of order: Later cells depend on variables created in earlier cells. Run from top to bottom.
  • .loc vs .iloc confusion: .loc[0:3] includes row 3 (4 rows), while .iloc[0:3] excludes row 3 (3 rows).
  • Missing outputs: If you see In [*]: next to a cell, it hasn't finished running. Wait for it to complete or re-run it.
  • No explanation: Where the question says "explain," write a comment. A blank answer or code-only answer will lose points.
