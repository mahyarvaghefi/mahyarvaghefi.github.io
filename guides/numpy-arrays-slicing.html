1.1 — What is NumPy?

NumPy (Numerical Python) is the foundational package for scientific computing in Python. It provides powerful tools for working with arrays — organized collections of numbers that can be manipulated efficiently.

The Big Picture

Everything in AI and machine learning — from ChatGPT to neural networks — processes data as matrices. Data goes in as matrices, computations happen on matrices, and results come out as matrices. NumPy is the foundation for all of this.

Why NumPy over Python Lists?

Speed

Adding 2 to a million numbers with a for loop? Takes a while. With NumPy? About one second.

Elegant Syntax

Treat the entire array as a single object. Say "add 2 to the whole thing" instead of looping element by element.

Array Types: Rank 1 vs Rank 2

TypeDescriptionCode ExampleShape
Rank 1 (1D) One-dimensional, like a vector np.array([1, 2, 3]) (3,)
Rank 2 (2D) Two-dimensional, like a matrix np.array([[1,2,3], [4,5,6]]) (2, 3)

Common Exam Pitfall — The Shape Tuple

(3,) — The trailing comma indicates one dimension.
(3, 4) — Two numbers means two dimensions.

That comma with nothing after it is Python's way of saying "this is a tuple with just one element."

1.2 — Array Slicing & Indexing

Slicing is one of the most important skills in NumPy. The basic structure is: array[row_index, column_index]

The Golden Rule of Slicing

What you put in determines what you get out. Both parts (row and column) can be either a specific number (index) or a range (like 0:3). The combination determines the output shape.

The Three Scenarios to Memorize

What You InputWhat You GetExampleResult Shape
Both numbers (index, index) Single value a[1, 2] scalar
One number, one range Rank 1 array (1D) a[2, 1:3] (2,)
Both ranges Rank 2 array (2D) a[0:3, 1:4] (3, 3)

What Does the Colon Mean?

: by itself means "everything" (beginning to end)
:3 from start up to (but not including) index 3
2: from index 2 to the end
1:4 from index 1 up to (but not including) index 4

Exam Alert — Range vs Index Changes Output Dimension

a[1:2, :] → Shape (1, 4) — 2D array (range used)
a[1, :] → Shape (4,) — 1D array (index used)

Same row, different shapes! Using a range preserves dimensionality; using an index reduces it.

1.3 — Element-wise Operations

With NumPy, you're not thinking about individual elements anymore — you're treating the entire array as a single object.

Broadcasting

When you add a number to an array, NumPy automatically applies the operation to every element. No loops required. This is called broadcasting.

Common Element-wise Operations

OperationSyntaxIn-placeDescription
Additiona + 2a += 2Add 2 to all elements
Subtractiona - 3a -= 3Subtract 3 from all
Multiplicationa * 2a *= 2Multiply all by 2
Divisiona / 4a /= 4Divide all by 4

In-place vs New Array

a + 5 creates a new array (original unchanged).
a += 5 modifies the array in place (slightly more memory-efficient for large arrays).

Key Takeaways

✓NumPy is the foundation of all data science and AI in Python — everything is matrices.
✓Shape tells dimensionality: (3,) is 1D, (3, 4) is 2D — watch for that trailing comma.
✓Slicing rule: index + index = scalar, number + range = 1D, range + range = 2D.
✓Colon means "everything": a[1, :] gets all columns of row 1.
✓Element-wise operations apply to the whole array at once — no loops needed.
!Common exam trap: a[1:2, :] vs a[1, :] have different shapes.

Code Cheat Sheet

# Import NumPy
import numpy as np

# Create arrays
b = np.array([1, 2, 3])          # 1D: shape (3,)
a = np.array([[1,2,3], [4,5,6]]) # 2D: shape (2, 3)

# Check shape
print(a.shape)                   # Output: (2, 3)

# Indexing
a[1, 2]                          # Single value
a[1, :]                          # Entire row 1 (1D)
a[:, 1]                          # Entire column 1 (1D)
a[0:2, 1:3]                     # Subarray (2D)

# Element-wise operations
a + 2                            # Add 2 to all elements
a *= 3                           # Multiply all by 3 (in place)
