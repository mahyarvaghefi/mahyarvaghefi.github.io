Section 1.1: What is NumPy?

NumPy (Numerical Python) is the foundational package for scientific computing in Python. It provides powerful tools for working with arrays‚Äîorganized collections of numbers that can be manipulated efficiently.

üí°

The Big Picture

Everything in AI and machine learning‚Äîfrom ChatGPT to neural networks‚Äîprocesses data as matrices. Data goes in as matrices, computations happen on matrices, and results come out as matrices. NumPy is the foundation for all of this.

üöÄ Why NumPy Over Python Lists?

1. Speed

Adding 2 to a million numbers with a for loop? Takes a while. With NumPy? About one second.

2. Elegant Syntax

Instead of looping through elements one by one, treat the entire array as a single object. Say "add 2 to the whole thing" instead of "add 2 to element 1, then element 2..."

Array Types: Rank 1 vs Rank 2

Type Description Code Example Shape
Rank 1 (1D) One-dimensional, like a vector np.array([1, 2, 3]) (3,)
Rank 2 (2D) Two-dimensional, like a matrix np.array([[1,2,3], [4,5,6]]) (2, 3)

‚ö†Ô∏è

Common Exam Pitfall: The Shape Tuple

(3,) ‚Äî The trailing comma indicates one dimension
(3, 4) ‚Äî Two numbers means two dimensions

That comma with nothing after it is Python's way of saying "this is a tuple with just one element."

Section 1.2: Array Slicing & Indexing

Slicing is one of the most important skills in NumPy. The basic structure is: array[row_index, column_index]

üîë

The Golden Rule of Slicing

What you put in determines what you get out. Both parts (row and column) can be either a specific number (index) or a range (like 0:3). The combination determines the output shape.

üìã The Three Scenarios to Memorize

What You Input What You Get Example Result Shape
Both numbers (index, index) Single value a[1, 2] scalar
One number, one range Rank 1 array (1D) a[2, 1:3] (2,)
Both ranges Rank 2 array (2D) a[0:3, 1:4] (3, 3)

üîπ What Does the Colon Mean?

: by itself means "everything" (from beginning to end)
:3 means from start up to (but not including) index 3
2: means from index 2 to the end
1:4 means from index 1 up to (but not including) index 4
‚ö†Ô∏è

Exam Alert: Range vs Index Changes Output Dimension!

a[1:2, :] ‚Üí Shape (1, 4) ‚Äî 2D array (range used)
a[1, :] ‚Üí Shape (4,) ‚Äî 1D array (index used)

Same row, different shapes! Using a range (1:2) preserves dimensionality; using an index (1) reduces it.

Section 1.3: Element-wise Operations

With NumPy, you're not thinking about individual elements anymore‚Äîyou're treating the entire array as a single object.

‚ú®

The Magic of Broadcasting

When you add a number to an array, NumPy automatically applies the operation to every element. No loops required. This is called broadcasting.

üìã Common Element-wise Operations

Operation Syntax In-place Version Description
Addition a + 2 a += 2 Add 2 to all elements
Subtraction a - 3 a -= 3 Subtract 3 from all elements
Multiplication a * 2 a *= 2 Multiply all elements by 2
Division a / 4 a /= 4 Divide all elements by 4
Remainder a % 2 a %= 2 Remainder when divided by 2

üí° Pro Tip: In-place vs New Array

a + 5 creates a new array (original unchanged)
a += 5 modifies the array in place (slightly more memory-efficient for large arrays)

Quick Reference: Key Takeaways

‚úì

NumPy is the foundation of all data science and AI in Python‚Äîeverything is matrices.

‚úì

Shape tells dimensionality: (3,) is 1D, (3, 4) is 2D‚Äîwatch for that trailing comma!

‚úì

Slicing rule: index + index = scalar, number + range = 1D, range + range = 2D.

‚úì

Colon means "everything": a[1, :] gets all columns of row 1.

‚úì

Element-wise operations apply to the whole array at once‚Äîno loops needed.

‚ö†Ô∏è

Common exam trap: a[1:2, :] vs a[1, :] have different shapes!

"If Python is your car, NumPy is the engine that makes everything go fast."

üìù Code Cheat Sheet

# Import NumPy

import numpy as np

# Create arrays

b = np.array([1, 2, 3])                     # 1D: shape (3,)

a = np.array([[1,2,3], [4,5,6]])          # 2D: shape (2, 3)

# Check shape

print(a.shape)                            # Output: (2, 3)

# Indexing

a[1, 2]                                   # Single value

a[1, :]                                   # Entire row 1 (1D)

a[:, 1]                                   # Entire column 1 (1D)

a[0:2, 1:3]                               # Subarray (2D)

# Element-wise operations

a + 2                                     # Add 2 to all elements

a *= 3                                    # Multiply all by 3 (in place)
